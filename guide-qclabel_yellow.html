🏷️ QC Tote Label Maker

Busbar A / Busbar B / MICA QC 라벨을 4"×4" PDF로 생성

💾 다운로드 (SharePoint) tote_label_maker_yellow.exe

⬇️ 설치 가이드 (최초 1회)

▼
1

위 다운로드 버튼 클릭

위쪽 파란 다운로드 버튼을 누르면 회사 SharePoint 페이지로 이동합니다.

2

회사 계정으로 로그인

FNS 회사 계정(이메일)으로 로그인하세요. 이미 로그인되어 있으면 이 단계는 건너뜁니다.

3

다운로드 진행

SharePoint에서 좌측 상단 다운로드 버튼을 클릭하여 tote_label_maker_yellow.exe 파일을 다운로드합니다.

💡 단일 EXE 파일입니다. 압축 해제 없이 원하는 폴더에 놓고 바로 실행하면 됩니다.
이름 저장 기능이 있어 EXE 파일 옆에 inspector_names.json 파일이 자동 생성됩니다.
4

처음 실행 시 Windows 보안 경고

EXE를 더블클릭하면 처음 한 번은 아래와 같은 경고가 뜹니다. 이건 정상이며, 코드 서명 인증서가 없는 EXE에 항상 나타납니다.

"More info"를 클릭하면 "Run anyway" 버튼이 나타납니다. 이걸 누르면 실행됩니다.

💡 한 번만 하면 됩니다: 처음 실행 후 이 경고가 다시 뜨지 않습니다.
5

그래도 실행이 막힌다면 (차단 해제)

Windows 스마트 스크린이 더 강하게 막아 "Run anyway" 버튼이 안 보이거나 실행이 계속 차단될 때가 있습니다. 이럴 때 파일 자체의 차단을 풀어주면 됩니다.

tote_label_maker_yellow.exe 우클릭 → "속성(Properties)" → 하단 "차단 해제(Unblock)" 체크 → "확인" 후 다시 실행하세요.

"차단 해제" 항목이 안 보이면 이미 차단이 풀린 상태이거나 처음 경고(4번) 단계에서 해결된 것이므로, More info → Run anyway로 실행하면 됩니다.

📖 사용 가이드

▼

EXE를 실행하면 아래와 같은 창이 열립니다. 각 항목을 채우고 Generate PDF를 누르면 4"×4" 라벨 PDF가 생성됩니다.

1

DATE — 날짜 선택

라벨에 인쇄될 날짜입니다. 기본값은 오늘 날짜로 자동 설정되며, 캘린더를 클릭해서 변경할 수 있습니다.

2

INSPECTOR — 검사자 이름 입력/선택

라벨에 인쇄될 검사자 이름입니다. 직접 타이핑하거나, 드롭다운을 열어 이전에 입력했던 이름을 선택할 수 있습니다.

💡 이름 자동 저장: 새 이름을 입력하고 PDF를 생성하면, 다음 실행 시 드롭다운에 자동으로 추가됩니다.
3

ITEM — 아이템 선택

라벨 종류를 선택합니다. 아이템에 따라 라벨 제목과 번호 표시가 달라집니다.

  • Busbar A — 라벨 제목: "Busbar A QC", 번호: 1-1 A 형식
  • Busbar B — 라벨 제목: "Busbar B QC", 번호: 1-1 B 형식
  • MICA — 라벨 제목: "MICA QC", 번호: 1-1 형식 (suffix 없음)
Busbar A QC
DATE: 06/12/2026
INSPECTOR: HANA KIM
A B C D
1-1 A
Busbar B QC
DATE: 06/12/2026
INSPECTOR: HANA KIM
A B C D
1-1 B
MICA QC
DATE: 06/12/2026
INSPECTOR: HANA KIM
A B C D
1-1
4

Set range — 세트 범위 설정

생성할 세트 범위를 입력합니다. 한 세트 = 8장 (1번~8번)입니다.

  • 예시: 1 ~ 10 — 1-1부터 10-8까지, 총 80장 생성
  • 예시: 3 ~ 3 — 3-1부터 3-8까지, 총 8장 생성
  • 예시: 5 ~ 7 — 5-1부터 7-8까지, 총 24장 생성
특정 번호만(아래 항목)에 값이 있으면 Set range는 무시됩니다.
5

특정 번호만 (선택사항)

일부 라벨만 다시 출력할 때 사용합니다. X-Y 형식으로, 쉼표 또는 줄바꿈으로 여러 개 입력 가능합니다.

  • 입력 예시: 3-6, 5-2, 8-1 → 3세트 6번, 5세트 2번, 8세트 1번 총 3장만 출력
  • 비워두면 → Set range에 설정한 전체 범위 출력
6

Generate PDF 클릭

버튼을 누르면 저장 위치를 묻는 대화창이 열립니다. 저장 경로를 선택하면 PDF 생성이 시작되고, 완료 후 자동으로 PDF가 열립니다.

💡 생성되는 파일명 예시:
전체 범위: Busbar_A_QC_1-10.pdf
특정 번호: Busbar_A_QC_3-6_5-2.pdf
라벨이 많을 때(80장 이상)는 생성에 수초 걸릴 수 있습니다. 진행률 바가 표시됩니다.

💡 알아두면 편한 것들

▼
  • 라벨 크기 — 각 페이지는 정확히 4"×4"로 설정됩니다. 4×4 라벨 프린터에서 바로 출력하세요.
  • A B C D 행 — 라벨 하단에 A B C D가 인쇄됩니다. 검사 항목에 동그라미 표시용입니다.
  • 이름 삭제 — 저장된 이름 목록에서 특정 이름을 지우려면 EXE 옆의 inspector_names.json을 메모장으로 열어 직접 편집하세요.
  • EXE 첫 실행 느림 — 처음 실행할 때 내부 파일 압축 해제 때문에 수초 걸릴 수 있습니다. 두 번째부터는 빠릅니다.
