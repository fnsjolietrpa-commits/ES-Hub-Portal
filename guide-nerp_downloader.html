📦 NERP IB Data Downloader

NERP Move Pallet Stock 데이터를 자동으로 다운로드하고 WMS와 비교

💾 다운로드 (SharePoint) NERP IB Data Downloader.zip

⬇️ 설치 가이드 (최초 1회)

▼
1

위 다운로드 버튼 클릭

위쪽 파란 다운로드 버튼을 누르면 회사 SharePoint 페이지로 이동합니다.

2

회사 계정으로 로그인

FNS 회사 계정(이메일)으로 로그인하세요. 이미 로그인되어 있으면 이 단계는 건너뜁니다.

3

다운로드 진행

SharePoint에서 좌측 상단 다운로드 버튼을 클릭하여 다운로드합니다.

4

압축 해제

다운로드된 IB Data Downloader.zip 파일을 우클릭 → "압축 풀기" 후 IB Data Downloader 폴더 안의 IB Data Downloader.exe를 실행합니다.

⚠️ 중요: 압축을 풀면 IB Data Downloader 폴더가 생성됩니다. 폴더 안에 _internal 폴더와 .exe 파일이 함께 있어야 합니다. exe 파일 단독으로 복사하면 실행되지 않습니다.
5

처음 실행 시 Windows 보안 경고

EXE를 더블클릭하면 처음 한 번은 아래와 같은 경고가 뜹니다. 이건 정상이며, 코드 서명 인증서가 없는 EXE에 항상 나타납니다.

"More info"를 클릭하면 "Run anyway" 버튼이 나타납니다. 이걸 누르면 실행됩니다.

💡 한 번만 하면 됩니다: 처음 실행 후 이 경고가 다시 뜨지 않습니다.
6

그래도 실행이 막힌다면 (차단 해제)

Windows 스마트 스크린이 더 강하게 막아 "Run anyway" 버튼이 안 보이거나 실행이 계속 차단될 때가 있습니다. 이럴 때 파일 자체의 차단을 풀어주면 됩니다.

IB Data Downloader.exe 우클릭 → "속성(Properties)" → 하단 "차단 해제(Unblock)" 체크 → "확인" 후 다시 실행하세요.

"차단 해제" 항목이 안 보이면 이미 차단이 풀린 상태이거나 처음 경고(5번) 단계에서 해결된 것이므로, 5번의 More info → Run anyway로 실행하면 됩니다.
7

.env 파일 설정

NERP IB Data Downloader 폴더 안에 .env 파일이 있습니다. 메모장으로 열어서 NERP 포털 로그인 정보를 입력하세요.

  • NERP_USERID — NERP 포털 아이디
  • NERP_PASSWORD — NERP 포털 비밀번호
  • WAREHOUSE_NUMBER — 창고 코드 (기본값: CHW1)
⚠️ .env 파일이 없으면 실행되지 않습니다. 배포 담당자에게 문의하세요.

📖 사용 가이드

▼

EXE를 실행하면 아래와 같은 창이 열립니다. Move Pallet Stock 데이터를 자동으로 다운로드하고, WMS와 팔렛·수량을 비교합니다.

1

WMS 아이디 / 비밀번호 입력

Username · Password에는 본인의 WMS(wms.itsscm.com) 로그인 아이디와 비밀번호를 입력합니다.

💡 Save ID/PW: 체크해두면 아이디·비밀번호가 PC에 저장되어 다음 실행 때 자동으로 채워집니다.
2

HBL Number 입력 (선택사항)

특정 HBL만 비교하고 싶을 때 사용합니다. 쉼표(,) 또는 줄바꿈으로 여러 개 입력 가능합니다.

  • 비워두면 — NERP에 있는 오늘자 전체 HBL을 비교합니다
  • 입력하면 — 입력한 HBL에 해당하는 데이터만 비교합니다
3

Run Script 클릭

"Run Script" 버튼을 누르면 자동으로 아래 순서로 진행됩니다.

  • 1단계 · 다운로드 — 포털에 자동 로그인 후, Move Pallet Stock 화면에서 오늘 날짜로 조회·엑셀 다운로드 (백그라운드, 화면 안 보임)
  • 2단계 · WMS 비교 — WMS에 로그인 후 HBL별로 팔렛 ID·아이템 코드·수량·Plant를 비교 (백그라운드)
  • 3단계 · 결과 표시 — 비교 완료 후 콘솔에 데이터 요약표와 비교 결과가 자동으로 표시됩니다
작업 중에는 Chrome 창이 화면 밖 백그라운드로 실행됩니다. 창이 안 보여도 정상입니다.
4

결과 확인 — NERP 데이터 요약표

비교가 끝나면 콘솔 상단에 NERP 데이터 요약표가 먼저 표시됩니다.

  • HBL — 선하증권 번호
  • ITEM CODE — 자재 코드 (HBL당 여러 개일 수 있음)
  • PLT 수 — 팔렛 총 개수
  • Total Qty — 전체 수량 합계
  • Storage Bin — 창고 보관 위치
5

결과 확인 — NERP↔WMS 비교 결과

요약표 아래에 HBL별 비교 결과가 표시됩니다.

  • [일치] NERP와 WMS 데이터가 완전히 일치
  • [확인필요] WMS에서 검색 불가 (이미 오더 컨펌된 건 등)
  • [불일치] 팔렛 ID · 아이템 코드 · 수량 · Plant 중 하나 이상 다름
[불일치]가 나오면 세부 내용(어떤 팔렛, 어떤 항목이 다른지)이 아래에 추가로 표시됩니다.
6

IB Order 결과 확인 팝업

모든 HBL이 일치하고 IB Order ID가 존재하면, WMS에서 자동으로 IB Order 검색까지 완료한 후 아래 팝업이 뜹니다.

  • 예 숨겨져 있던 Chrome 창이 전체화면으로 나타나고, 방금 검색한 IB Order 결과를 WMS에서 바로 확인할 수 있습니다
  • 아니오 Chrome 브라우저를 종료합니다
💡 "예"를 선택하면 WMS 브라우저가 열립니다. 확인 후 브라우저를 직접 닫으면 프로그램이 완전히 종료됩니다.

💡 알아두면 편한 것들

▼
  • NERP 로그인 느릴 때 — .env 파일의 아이디·비밀번호가 맞는지 확인하세요. 포털 로딩이 느린 경우 자동으로 기다립니다.
  • HBL이 NERP에 없다는 메시지 — 오늘 날짜에 GR 처리된 건이 아닌 경우 NERP 조회 결과에 없을 수 있습니다.
  • [확인필요] 메시지 — 해당 HBL이 WMS에서 이미 오더 컨펌(Order Confirm) 처리된 건일 가능성이 높습니다. WMS에서 직접 확인하세요.
  • 비교 결과가 안 나올 때 — WMS 아이디·비밀번호를 확인하세요. 로그인 실패 시 콘솔에 오류 메시지가 표시됩니다.
