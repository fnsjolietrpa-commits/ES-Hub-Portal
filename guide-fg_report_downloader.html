한국어 가이드

⬇️ 설치 (최초 1회)

▼
1

위 다운로드 버튼 클릭

파란 버튼을 누르면 회사 SharePoint 페이지로 이동합니다.

2

회사 계정으로 로그인

FNS 회사 이메일로 로그인하세요. 이미 로그인되어 있으면 자동으로 건너뜁니다.

3

파일 다운로드

SharePoint 페이지 좌측 상단 다운로드 버튼을 클릭하여 zip 파일을 받습니다.

4

압축 해제

FG Report Downloader.zip 우클릭 → "압축 풀기" → 생성된 FG Report Downloader 폴더 안의 FG Report Downloader.exe를 실행합니다.

⚠️ 중요: 폴더 안에 _internal 폴더와 .exe 파일이 함께 있어야 합니다. exe 파일만 단독으로 복사하면 실행되지 않습니다.
5

처음 실행 시 Windows 보안 경고

처음 실행하면 아래와 같은 파란 경고 화면이 뜰 수 있습니다. 코드 서명 인증서가 없는 EXE에는 항상 나타나며 정상입니다.

"More info"를 클릭하면 "Run anyway" 버튼이 나타납니다. 클릭하면 실행됩니다.

💡 최초 1회만: 한 번 실행 후 이 경고는 다시 뜨지 않습니다.
6

그래도 막힌다면 — 파일 차단 해제

"Run anyway" 버튼이 보이지 않거나 계속 차단될 경우, 파일 속성에서 직접 차단을 해제하세요.

FG Report Downloader.exe 우클릭 → 속성(Properties) → 하단 "차단 해제(Unblock)" 체크 → 확인 후 다시 실행.

"차단 해제" 항목이 없으면 이미 차단이 풀린 상태입니다. 5번의 More info → Run anyway로 실행하세요.
7

.env 파일 설정

FG Report Downloader 폴더 안의 .env 파일을 우클릭 → 다른 앱으로 열기 → 메모장으로 열어 아래 항목을 입력합니다.

  • WMS_ID WMS 로그인 아이디
  • WMS_PW WMS 비밀번호
  • SAVE_DIR 리포트를 저장할 폴더. 비워두면 다운로드 폴더에 자동 저장.
💡 SAVE_DIR 설정법: 파일 탐색기에서 원하는 폴더를 열고, 상단 주소창을 클릭 → Ctrl+A → Ctrl+C로 경로를 복사해 SAVE_DIR= 뒤에 붙여넣기.
⚠️ WMS_ID와 WMS_PW가 없으면 실행되지 않습니다. 담당자에게 문의하세요.

English Guide

▼

⬇️ Installation (first time only)

▼
1

Click the download button above

The blue button will take you to the company SharePoint page.

2

Sign in with your company account

Log in with your FNS company email. If you're already signed in, this step is skipped automatically.

3

Download the file

On the SharePoint page, click the Download button in the top-left to download the zip file.

4

Unzip the file

Right-click FG Report Downloader.zip → "Extract All". Open the extracted folder and run FG Report Downloader.exe.

⚠️ Important: The _internal folder and the .exe file must stay together. Do not copy the .exe by itself — it won't run without the _internal folder next to it.
5

Windows security warning (first run)

The first time you run the EXE, Windows may show a blue warning screen. This is normal for unsigned EXE files.

Click "More info" — a "Run anyway" button will appear. Click it to proceed.

💡 One time only: This warning will not appear again after the first run.
6

If the app is still blocked — Unblock the file

If "Run anyway" is not shown, unblock the file from its properties.

Right-click FG Report Downloader.exe → Properties → check "Unblock" at the bottom → click OK, then run again.

If "Unblock" is not visible, the file is already unblocked. Use More info → Run anyway from step 5.
7

Configure the .env file

Inside the FG Report Downloader folder, right-click the .env file → Open with → Notepad and fill in your credentials.

  • WMS_ID Your WMS login username
  • WMS_PW Your WMS password
  • SAVE_DIR Folder to save the reports. Leave blank to use your Downloads folder automatically.
💡 Setting a custom SAVE_DIR: In File Explorer, open the folder you want, click the address bar at the top, press Ctrl+A then Ctrl+C to copy the path, and paste it after SAVE_DIR=.
⚠️ The app will not run without WMS_ID and WMS_PW. Contact your administrator if you don't have a .env file.
