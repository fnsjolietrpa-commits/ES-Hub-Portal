🖨 PDF Merger + 자동 출력

픽업 PDF를 병합하고 SEAL # 스탬프 후 양면 자동 출력

💾 다운로드 (SharePoint) PDF Merger.zip

⬇️ 설치 가이드 (최초 1회)

▼
1

위 다운로드 버튼 클릭

파란 다운로드 버튼을 누르면 SharePoint 폴더로 이동합니다. PDF Merger 폴더를 다운로드하세요.

2

압축 해제 후 폴더 배치

다운로드된 ZIP 파일을 우클릭 → "압축 풀기"합니다. 생성된 PDF Merger 폴더를 원하는 위치에 놓으세요.

⚠️ _internal 폴더를 절대 삭제하지 마세요!
폴더 안에 PDF Merger.exe와 _internal 폴더가 반드시 함께 있어야 합니다. _internal을 지우거나 .exe만 따로 복사하면 실행되지 않습니다.
3

처음 실행 시 Windows 보안 경고

EXE를 더블클릭하면 처음 한 번은 아래와 같은 경고가 뜹니다. 이건 정상입니다.

"More info"를 클릭하면 "Run anyway" 버튼이 나타납니다.

💡 한 번만 하면 됩니다: 처음 실행 후 이 경고가 다시 뜨지 않습니다.

📖 사용 가이드

▼

EXE를 실행하면 아래와 같은 창이 열립니다. 픽업 PDF들을 병합하고, SEAL # 스탬프를 찍은 후 양면으로 자동 출력합니다.

1

입력 선택 — ZIP 또는 폴더

픽업 PDF들이 담긴 ZIP 파일 또는 폴더를 선택합니다.

  • ZIP 선택 — ZIP 파일을 직접 선택. 자동으로 압축 해제 후 병합
  • 폴더 선택 — PDF가 들어있는 폴더를 선택
💡 PDF는 파일명 기준으로 자동 정렬(자연 정렬)됩니다. 파일명 앞에 번호를 붙여두면 순서대로 처리됩니다.
2

프린터 선택

드롭다운에서 사용할 프린터를 선택합니다. 기본 프린터가 자동으로 선택됩니다. 목록이 안 뜨면 ↻ 새로고침 버튼을 누르세요.

3

SEAL # 입력 (선택사항)

4자리 시작 번호를 입력하면 각 PDF에 SEAL # 스탬프가 자동으로 찍힙니다.

  • 입력 예: 1001 — 1번 PDF → FNS 1001, 2번 PDF → FNS 1002, ...
  • 비워두면 — SEAL 스탬프 없이 그대로 병합
스탬프 위치: 픽업 PDF의 SPECIAL INSTRUCTION 섹션 빈칸에 자동으로 삽입됩니다.
4

버튼 클릭 — 병합만 / 병합+출력

버튼을 누르면 확인 팝업이 먼저 뜹니다. SEAL # 정보를 확인 후 OK를 누르면 시작됩니다.

  • ▶ 병합만 — 픽업 PDF + CHECK LIST를 병합해서 Downloads 폴더에 저장
  • 🖨 병합+출력 — 병합 후 지정 프린터로 양면(긴 면 기준) 출력, 완료 후 임시 파일 자동 삭제
💡 CHECK LIST는 별도 선택 불필요: EXE 안에 이미 포함되어 있어서 자동으로 사용됩니다.
5

결과 확인

로그 창에서 진행 상황을 실시간으로 확인할 수 있습니다.

  • 병합만 — Downloads\merged_날짜시간.pdf로 저장됨
  • 병합+출력 — 출력 완료 후 임시 PDF 자동 삭제됨 (Downloads에 파일 남지 않음)

💡 알아두면 편한 것들

▼
  • _internal 폴더 삭제 금지 — SumatraPDF(양면 출력 엔진)와 CHECK LIST.pdf가 들어있습니다. 지우면 실행 자체가 안 됩니다.
  • 출력이 단면으로 나올 때 — 프린터가 양면을 지원하는지 확인하세요. 이 프로그램은 항상 양면(긴 면 기준)으로 전송합니다.
  • CHECK LIST 내용 변경 필요할 때 — 담당자에게 요청하거나 새 EXE를 재배포해야 합니다 (CHECK LIST.pdf는 EXE 내부에 포함됨).
  • SEAL # 4자리가 아니면 — 입력이 차단됩니다. 숫자 4자리만 입력 가능합니다.
  • PDF 순서가 잘못됐을 때 — 파일명 앞에 01_, 02_ 같이 번호를 붙이면 원하는 순서로 정렬됩니다.
문의/오류 보고: 담당자에게 연락주세요
