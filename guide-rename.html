📦 선적서류 파일명 컨버터

선적서류 PDF/Excel 파일명을 PO + HBL 형식으로 자동 변환

💾 다운로드 (SharePoint) 선적서류_파일명변경.zip · 약 68MB

⬇️ 설치 가이드(소요 시간 2분)

1

위 다운로드 버튼 클릭

위쪽 파란색 다운로드 버튼을 누르면 회사 SharePoint 페이지로 이동합니다.

2

회사 계정으로 로그인

FNS 회사 계정(이메일)으로 로그인하세요. 이미 로그인되어 있으면 이 단계는 건너뜁니다.

3

다운로드 진행

SharePoint에서 좌측 상단 다운로드 버튼을 클릭하여 다운로드합니다.

4

압축 해제

다운로드된 선적서류_파일명변경.zip 파일을 우클릭 → "압축 풀기"후 선적서류_파일명변경.exe을 실행합니다.

5

처음 실행 시 Windows 디펜더 경고

EXE를 더블클릭하면 처음 한 번은 아래와 같은 경고가 뜹니다. 이건 정상이며, 코드 서명 인증서가 없는 EXE에 항상 나타납니다.

"More info"를 클릭하면 "Run anyway" 버튼이 나타납니다. 이걸 누르면 실행됩니다.

💡 한 번만 하면 끝: 처음 실행 후엔 이 경고가 다시 뜨지 않습니다.

🚀 사용 가이드

EXE를 실행하면 아래와 같은 창이 열립니다.

1

대상 폴더 선택

오른쪽 위 "폴더 선택" 버튼을 클릭해서 이름을 바꿀 선적서류(PDF/Excel)들이 들어 있는 폴더를 선택합니다.

서류가 들어있는 "폴더"를 선택하는 것이기 때문에 폴더 내 서류가 보이지않더라도 정상적인 동작입니다.
2

검증 테이블 입력 (선택사항)

WMS/시스템에서 뽑은 PO 리스트가 있으면 함께 입력하면 PDF 내용과 자동 대조해서 누락/오류를 잡아줍니다.

  • 파일 선택 — 엑셀 파일(.xlsx)을 직접 지정하거나
  • 붙여넣기 — 엑셀에서 표 영역을 복사해서 아래 칸에 붙여넣거나
  • 비워두기 — 표 없이 그냥 이름 변경만 실행할 수도 있음

표는 반드시 다음 6개 컬럼이 있어야 합니다: PO #, HBL #, CONSIGNEE, MATERIAL CD, MATERIAL, QTY

3

실행 버튼 클릭

"실행" 버튼을 누르면 확인 다이얼로그가 뜹니다. "예"를 누르면 처리 시작.

처리 중에는 아래 진행률 바가 움직입니다. 파일 수와 크기에 따라 보통 10초~1분 정도 걸립니다.

4

결과 확인

처리 결과 영역에 표가 표시됩니다.

  • ⭕ PO/HBL/코드/수량 모두 일치 — 문제 없음
  • ⭕ (HBL 추정) HBL이 PDF에 없어서 PO+수량으로 매칭 — 수동 확인 권장
  • ❌ 파일 없음 표에는 있지만 폴더에 PDF가 없음
  • ❌ 수량 불일치 표의 수량이 PDF에 나타나지 않음
  • ❌ 코드 다름 아이템 코드가 표와 다름

불일치 항목은 위쪽에, 일치 항목은 아래쪽에 정렬됩니다. 처리 폴더 안에 자동으로 로그 파일(rename_log_YYYYMMDD_HHMMSS.txt)도 저장되어 변경 내역을 추후 확인 가능합니다.

❓ 자주 묻는 질문

Q

왜 파일명이 변경된 후에는 되돌릴 수 없나요?

윈도우 파일 이름 변경은 즉시 적용됩니다. 도구는 폴더 안에 rename_log_*.txt를 자동 저장하므로, 원본 이름이 필요하면 그 로그를 참고하세요. 처음 사용 시엔 폴더를 먼저 복사해두고 테스트하는 걸 권장합니다.

Q

"❌ 수량 불일치" 가 나오는데 실제로 PDF에 수량이 있어요

PDF 텍스트 추출 한계로 가끔 표 안의 숫자들이 붙어버려서 인식이 안 되는 경우가 있습니다. 로그 파일에서 정확한 사유를 확인하고, 필요시 수동으로 확인해주세요.

Q

같은 PO로 시작하는 파일이 여러 개 있을 때 충돌은?

같은 PO+HBL+문서종류가 여러 파일이면 자동으로 _2, _3 처럼 번호가 붙습니다. 덮어쓰지 않으니 안전합니다.
